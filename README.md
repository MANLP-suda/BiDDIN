# Requirements
Python version >= 3.6

Pytorch version >= 1.6.0

Sklearn version >= 1.0.1

Scipy version >= 1.6.1
# Running
Run our code with IEMOCAP dataset
```
"python my_IEMOCAP.py"
```
Run our code wit AVEC dataset
```
"python my_AVEC.py"
```
